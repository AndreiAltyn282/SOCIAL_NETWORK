# 🤝 Как помочь проекту ProSocial

Спасибо, что хотите внести вклад в ProSocial! Мы рады любым идеям, баг-репортам и пулл-реквестам.

## 🐛 Сообщение об ошибке

1. Проверьте, что ошибка не была уже описана в [Issues](https://github.com/AndreiAltyn282/SOCIAL_NETWORK/issues)
2. Создайте новый Issue с подробным описанием:
   - Шаги воспроизведения
   - Ожидаемое поведение
   - Скриншоты (если есть)
   - Версия окружения (браузер, ОС, версия Python)

## 💡 Предложение новой фичи

1. Проверьте, что фича не была уже предложена
2. Создайте новый Issue с меткой `enhancement`
3. Опишите фичу, её преимущества и пример реализации

## 🔧 Пулл-реквест

1. **Форкните** репозиторий
2. **Клонируйте** свой форк
3. Создайте **ветку** для изменений:
   ```bash
   git checkout -b feature/my-feature
